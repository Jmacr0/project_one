ReadMe search works just like web search, but to search a user's favourite books. The user provides input data to filter results. The input includes; Keywords, Title, Author, and Genre. ReadMe provides the book image, title, author and plot summary. Generally, eight books are displayed at a time and paginated to five pages. A total of forty books per search will be displayed. Upon executing a new search, the previous search results are cleared.

The main features are Genre and Random; With Genre the drop down starts as empty; When the user begins to search, the Genre populated will be saved in local storage. When a user exits a search and reopens, it will repopulate with previous genre. Random: The ReadMe application includes a random feature which allows the user to search for a list of random books. This works randomly selecting a word and using the returned word as a keyword input.
